# 🧩 Personal Portfolio

> A modern, responsive personal portfolio built with a **Bento Grid** layout concept. Clean, minimal, and interactive.

## ✨ Features

- 🧱 **Bento Grid Layout** modern mosaic style card layout for showcasing content
- 📱 **Mobile Responsive** fully optimized across all screen sizes
- 🎨 **shadcn/ui Components** consistent and accessible UI components
- ⚡ **Next.js App Router** fast, server side rendering out of the box
- 🔒 **TypeScript** fully typed for better DX and maintainability

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:
- Node.js `>= 18`
- npm / yarn / pnpm / bun

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/your-portfolio.git

# Navigate to the project directory
cd your-portfolio

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
